# Placeholder screenshots

Replace these placeholder files with actual screenshots before Play Store submission.

Required screenshot sizes:
- Phone: 1080 x 1920 px
- TV: 1280 x 720 px
- 7-inch tablet: 1200 x 1920 px
- 10-inch tablet: 2000 x 1200 px

Minimum 2 screenshots per device type, maximum 8.
